# SEO Implementation Checklist

Use this checklist to ensure all pages have comprehensive SEO, AEO, and GEO optimization.

## ✅ Basic Meta Tags

- [ ] Unique, descriptive title tag (50-60 characters)
- [ ] Compelling meta description (150-160 characters)
- [ ] Relevant keywords (natural, not overstuffed)
- [ ] Author meta tag
- [ ] Language meta tags (en-CA for English, fr-CA for French)
- [ ] Robots meta tags (index, follow)

## ✅ Open Graph Tags (Social Sharing)

- [ ] og:title
- [ ] og:description
- [ ] og:type (website, article, etc.)
- [ ] og:url (canonical URL)
- [ ] og:image (1200x630px recommended)
- [ ] og:image:width
- [ ] og:image:height
- [ ] og:image:alt
- [ ] og:site_name
- [ ] og:locale (en_CA, fr_CA)
- [ ] og:locale:alternate (for bilingual pages)

## ✅ Twitter Card Tags

- [ ] twitter:card (summary_large_image)
- [ ] twitter:title
- [ ] twitter:description
- [ ] twitter:image
- [ ] twitter:image:alt

## ✅ Technical SEO

- [ ] Canonical URL tag
- [ ] Alternate language links (hreflang)
- [ ] Proper HTML lang attribute
- [ ] Mobile-responsive viewport meta tag
- [ ] Charset declaration (UTF-8)

## ✅ Geographic Targeting

- [ ] geo.region (CA-QC)
- [ ] geo.placename
- [ ] geo.position (coordinates)
- [ ] ICBM coordinates

## ✅ Structured Data (JSON-LD)

- [ ] Appropriate schema.org type
- [ ] Organization/Service/Article schema
- [ ] Complete information (name, description, URL)
- [ ] Valid JSON-LD format

## ✅ Content Optimization

- [ ] H1 tag with primary keyword
- [ ] Proper heading hierarchy (H1, H2, H3)
- [ ] Descriptive alt text on all images
- [ ] Internal linking to related pages
- [ ] Natural keyword usage in content
- [ ] Comprehensive, high-quality content

## ✅ AEO/GEO Optimization

- [ ] Clear, structured content
- [ ] Semantic HTML markup
- [ ] FAQ sections (where applicable)
- [ ] Natural language optimized for AI
- [ ] Comprehensive topic coverage
- [ ] Structured data for context

## ✅ Performance

- [ ] Optimized images (compressed, appropriate size)
- [ ] Fast page load times
- [ ] Mobile-friendly design
- [ ] Accessible markup

## ✅ robots.txt

- [ ] robots.txt file exists
- [ ] Allows search engine crawlers
- [ ] Allows AI crawlers (GPTBot, Claude, etc.)
- [ ] Sitemap reference included
