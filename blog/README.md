# Blog

This folder contains blog posts and articles for the Prime Cybersecurity website.

## Structure

Blog posts can be organized by date or category. Each post should be an HTML file or markdown file that can be converted to HTML.

## Example Structure

```
blog/
├── 2025/
│   ├── 01/
│   │   └── post-title.html
│   └── 02/
│       └── another-post.html
└── README.md
```
